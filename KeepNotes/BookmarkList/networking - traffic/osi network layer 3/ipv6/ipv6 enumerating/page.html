Local or Remote attacks.
Mainly looking for opportunities in misconfigured firewalls/filters (IPv4 set up, but not IPv6).

Local Enumeration:
# Ping6 to multicast and receive responses. List with ip -6 neigh
# ping6 -c 5 ff02::1%eth0 >/dev/null
# ip -6 neigh

Remote Enumeration:
Must rely on DNS, error message content, HTTP/JS content, etc.
Make note of IPv6 addresses that you can gather from IPv4 known addresses (AAAA record type is for IPv6)
# dig +short IN AAAA www.google.com

Script options:
Passive listening
# detect-new-ip6 eth0 (may be renamed to atk6-detect-new-ip6)

Scanning IPv6:
Native tools like nmap
nmap -6 -sS -sC fc00:XXXX
