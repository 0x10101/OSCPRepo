Losing Your Keys
Access to AWS services is done by keys (accessKey and SecretKey). Permissions are highly granular. Of course the permissions you get to AWS differs based on the keys that you use, as you’d expect. One vulnerability to consider is losing your keys. If AWS Keys (accessKey and secretKey) are ever disclosed then bad things can occur – an attacker would obviously gain all of the privileges that those keys offer.
