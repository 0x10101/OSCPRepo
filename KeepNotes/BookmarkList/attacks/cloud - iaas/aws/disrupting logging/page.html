In regards to Cloudtrail, here are some commands to interact with logs
aws cloudtrail delete-trail --name CloudTrail
aws cloudtrail stop-logging --name CloudTrail
aws cloudtrail update-trail --name CloudTrain --no-is-multi-region-trail --no-include-global-service-events
aws s3 rb --force s3://mu-cloudtrail-bucket
