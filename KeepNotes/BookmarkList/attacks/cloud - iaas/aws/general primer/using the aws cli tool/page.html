aws s3 ls s3://<bucket> --no-sign-request List Objects in a bucket
aws s3 cp s3://<bucket>/item . --no-sign-request Copy object from bucket to local
aws s3 cp item s3://<bucket>/item --no-sign-request Upload object to bucket from local
aws s3 api get-bucket-acl --bucket <bucket> --no-sign request Get bucket ACL
  can append ?policy to the path to query the bucket policy
aws s3 api get-object-acl --bucket <bucket> --key item --no-sign-request Get ACL for specific object
