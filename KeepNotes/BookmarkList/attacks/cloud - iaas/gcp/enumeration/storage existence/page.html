Google Storage is a service that provides static file hosting within resources known as "buckets". The AWS S3 buckets and Google Storage buckets are similar across the two clouds.

Google Storage bucket permission policies can get very fine-graned.

One tool to help enumerate is [GCPBucketBrute](https://github.com/RhinoSecurityLabs/GCPBucketBrute)

Instead of using "gsutil" (Google Storage Utility) CLI program, it was found to be a lot faster to hit each HTTP endpoint to check for existence. The tool will generate permutations based on keyword argument and make HEAD requests.

Each HEAD request will be made to:
https://www.googleapis.com/storage/v1/b/BUCKET_NAME

Any HTTP code except for 404 or 400 is an indicator that the bucket exists.
