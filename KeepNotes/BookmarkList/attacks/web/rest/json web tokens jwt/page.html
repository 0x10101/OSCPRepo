Analyze the access token: One of the most important vulnerability checking must be done here.  
Can use extensions for burp or web tools like jwt.io

A JSON token is comprised of 3 parts, separated by periods.
Header, Payload, and Signature.

Header contains Algorithm and Type
Payload contains the username, password, and other data
Signature is a hashed value of the secret

By generating new tokens, you can test for session and access issues.
