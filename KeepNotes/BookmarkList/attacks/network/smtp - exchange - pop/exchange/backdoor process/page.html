An example to backdoor Exchange

Process
1. Identify a non-protected security group with local admin rights on one or more Exchange Servers
2. Grant Authenticated Users full control over this security group
3. Change the owner of the group to an Exchange server
4. Denu Read Permissions on this group to the Everyone principal

Execution
1. Regain access to the Active Directory domain as any user
2. Add your current user to the backdoored security group
3. Use your new local admin rights on an Exchange server to execute commands as the SYSTEM user on that computer
4. Abuse the rights Exchange Trusted Subsystem has over the domain object (i.e. WriteDacl)
