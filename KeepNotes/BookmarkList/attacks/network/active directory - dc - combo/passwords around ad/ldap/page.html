Hunting for Passwords
 
There are 3-4 fields that seem to be common in most AD schemas,

UserPassword
UnixUserPassword
unicodePwd
msSFU30Password.

On a surprising number of tests we find one or more of these fields are populated with ACTUAL passwords. They are sometimes obfuscated by converting to the ASCII decimal equivalents but that is nothing that “man ascii” can’t help you with.
