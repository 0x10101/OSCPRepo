Two takeover primitives are
  • Forcing a password reset
  • Targeting Kerberoasting through SPN modification (for credential recovery)

Additional rights that we care about:
  • WriteProperty to all properties
  • WriteProperty to the servicePrincipalName (SPN)
  • All extended rights
    • User-Force-Change-Password

These rights are abusable through
  • Set-DomainObjectOwner
  • Set-DomainUserPassword
