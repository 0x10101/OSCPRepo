DS_CONTROL_ACCESS
Access mask bit that grants privs that aren't easily expressed in the access mask.
If the ObjectActType of an ACE with CONTROL_ACCESS set is the GUID of a confidential property or property set, then this bit controls read access to that property (eg, in the case of LAPS)
 

With Extended Rights
If the ObjectActType GUID matches a registered extended-right GUID, then CONTROL_ACCESS grants that particular "controll access right" (eg, User-Force-Change-Password on user objects, or DS-Replication-Get-Changes and DS-Replication-Get-Changes-All on the domain object itself)
