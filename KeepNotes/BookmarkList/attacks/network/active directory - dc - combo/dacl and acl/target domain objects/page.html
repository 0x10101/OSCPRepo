Main takeover primitive is
  • Granting a user domain replication rights (for DCSync)
    • Or someone who currently has DCSync rights

Additional rights that we care about:
  • WriteDacl (so we can grant a principal DCSync rights)

These rights are abusable through
  • Add-DomainObjectAcl
  • Or Explicit
    • DS-Replication-Get-Changes
    • DS-Replication-Get-Changes-All
