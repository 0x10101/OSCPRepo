ACLs, DACLs, and SACLs
Access Control List (ACL) is basically shorthand for the DACL/SACL superset
An onject's Discretionary Access Control List (DACL) and System Access Control List (SACL) are ordered collections of Access Control Entries (ACEs)
  DACL specifies what principals/trustees have what right rights over the object
  SACL allows for auditing of acess attempts to an object
