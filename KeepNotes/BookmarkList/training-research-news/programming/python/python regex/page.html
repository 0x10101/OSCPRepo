Substitutions
Regex is not always the fastest implementation, consider text.replace()
'Fastest' text.replace() method:
chars = \\`*_{}[]()>#+-..!$"
for c in chars:
    text = text.replace(c, "\\" + c)

Compiling a regex string is good if the string will be used multiple times, otherwise it creates overhead
rx = re.compile('([&#])')
text = rx.sub(r'\\\1', text)

Find & Replace All
import re
regex = re.compile('([a-zA-Z]\"[a-zA-Z])')
myfile = "foo's bar'
myfile2 = regex.sub(lambda m: m.group().replace('"'."%",1), myfile)
print(myfile2)

m is a match object, get the group (matched substring o"s) and replace " with %

Match vs Search
Match checks ONLY at the beginning of the string
Search checks ANYWHERE in the string


Regex Flags
re.A ASCII-only matching instead of full unicode matching
re.DEBUG Display debug information about the compiled expression
re.I Perform case-insensitive matching
re.L make \w, \W, \b, \B and case-insensitive matching dependent on the current locale
re.M Multiline, '^' matches at beginning of string and beginning of each line (following newline). '$' matches end of string and end of each line
re.S Make '.' match any character at all, including newline. without this flag, '.' will match anything except a newline
re.X Verbose
