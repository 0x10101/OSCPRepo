Documents and metadata can provide a lot of useful OSINT. Document types that may contain a lot of metadata include

pdf
doc, dot, docx
xls, xlt, xlsx
ppt, pot, pptx
jpg, jpeg
html, htm

A great built-in tool to help gather documents is wget:
-nd no directories (place all files in a specified directory)
-r recurse
-P <dir> place files in this directory
-R/A restrict or allow file types or patterns
# wget -nd -r -R htm,html,php,asp,aspx,cgi -P /target/dir <target_domain>
