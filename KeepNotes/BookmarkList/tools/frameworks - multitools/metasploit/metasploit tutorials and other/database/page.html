Default configuration file is in /usr/share/metasploit-framework/config/database.yml
So if you're ever disconnected, you can reconnect using that config file

db_connect <string> Connects to a database
db_disconnect Disconnects
db_driver Selects database type
db_status Displays the status of the db
db_exports Exports db contents to a file
db_import import from nmap, or other XML exports from tools
db_nmap run Nmap and import data into DB



Can manually enter data with
hosts --add <>
services --add <options>
