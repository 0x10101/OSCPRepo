Monitor and Alert
[Security_Monkey](https://github.com/Netflix/security_monkey) | Monitors AWS and GCP accounts for policy changes and alerts on insecure configurations

Automated Security
[aardvark](https://github.com/Netflix-Skunkworks/aardvark) | multi-account AWS IAM Access Advisor API
-> [RepoKid](https://github.com/Netflix/Repokid) | Uses Aardvark to remove permissions granting access to unused services from the inline policies of IAM roles in an AWS account

Scanning and Remediation
[Zeus](https://github.com/DenizParlak/Zeus) | Auding & Hardening for EC2, S3, CloudTrail, CloudWatch, KMS based on CIS AWS benchmark

[AWS Inspector](https://aws.amazon.com/inspector/) | Automated security assessment service
