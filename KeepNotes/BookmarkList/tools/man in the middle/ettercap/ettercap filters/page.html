Can alter traffic on the fly using custom rules.
Need to be written with etterfilter:
# etterfilter filerName -o filterName.ef

Loaded with:
# ettercap -F filterName.ef


Some verbs are:
search(where,what) search the "where" for the "what". Where can be DATA.data or DECODED.data.
regex(where,regex) replacement for search() with regular expressions
replace(what,with) Replace using DATA.data or DECODED.data
drop() Drop the current packet
inject(filename) inject the contents of a named file as a valid TCP or UDP payload; commonly follows drop() (eg client requests a file, drop the response and inject your own)
kill() terminate the connection with RST or ICMP unreachable
exec(command) execute the filesystem command
exit() stops the filter execution for the current packet
