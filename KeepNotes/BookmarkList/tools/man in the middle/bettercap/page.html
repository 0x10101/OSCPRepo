[Bettercap](https://www.bettercap.org) | Extensible MitM platform with a lot of built-in features (proxy server, cred sniffer, DNS proxy, more)

BetterCap supersedes Ettercap

Warning: without specifying, Bettercap targets all devices on the LAN, so specify a target list with -T, specify one with -G or exclude hosts on the LAN with --ignore
Bettercap's modules are easy to use and create.
Bettercap offers more flexibility and simpler output.
Bettercap won't contextualize certificates in SSL/HTTP, which Ettercap does.
Bettercap cannot modify the output HTTP requests from clients, which Ettercap can do (but it is difficult).

-L Sniff packets
--sniffer-output Write packets to pcap
--sniffer-filter "<>" sniff based on a filter
--custom-parser "<>" sniff and display based on regex
--proxy sniff and sent through Bettercap proxy server
--dns <file> use the text file to spoof DNS responses

Bettercap popular custom modules (https://github.com/evilsocket/bettercap-proxy-modules)
androidpwn.rb execute command exploiting Android addJavaScriptInterface vulnerability in Android < 4.2
osxsparkle.rb execute custom OS X Mach-O executable using Sparkle Updater vulnerability
beefbox.rb inject a beef hook
replace_images.rb replace images with specified alternative
replace_file.rb replace files downloaded with a specified alternative
sshdowngrade.rb implements an ssh 2.x to 1.x downgrade attack
mssqlauth.rb downgrade MSSQL connections to capture credentials
